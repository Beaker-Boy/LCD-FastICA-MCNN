# 灵活信号处理管道使用说明

## 功能概述

现在您可以在上传信号文件后，**按顺序选择最多 4 种处理方法**对该文件进行处理。系统会依次应用您选择的所有方法，最终将处理结果用于模型训练。

## 可用处理方法

目前支持以下 3 种方法：
- **None** - 跳过此步骤
- **LCD** - 局部特征尺度分解（生成 10 个 ISC 分量）
- **FastICA** - 快速独立成分分析（从多通道信号中提取 2 个独立成分）

## 使用步骤

### 1. 选择信号文件
点击"浏览"按钮选择您的 `.npy` 格式振动信号文件。

### 2. 设置基本参数
- 采样率 (Hz)：默认 20000
- 最大采样点数：默认 9142857
- 选择标签：A-J

### 3. 配置处理方法（关键！）

在"信号处理方法配置"区域，您可以按顺序配置最多 4 个处理步骤：

**示例配置 1：标准流程**
```
步骤 1: LCD
步骤 2: FastICA
步骤 3: None
步骤 4: None
```
效果：先进行 LCD 分解（10 通道），再进行 FastICA 分离（2 通道）

**示例配置 2：仅 LCD**
```
步骤 1: LCD
步骤 2: None
步骤 3: None
步骤 4: None
```
效果：只进行 LCD 分解（10 通道）

**示例配置 3：仅 FastICA**
```
步骤 1: FastICA
步骤 2: None
步骤 3: None
步骤 4: None
```
效果：直接对原始信号进行 FastICA（单通道，会提示跳过）

**示例配置 4：无处理**
```
步骤 1: None
步骤 2: None
步骤 3: None
步骤 4: None
```
效果：使用原始信号（1 通道）

### 4. 添加至批处理列表
配置完成后，点击"添加至批处理列表"按钮。

### 5. 开始处理
点击"开始批处理和训练"，系统将：
1. 按照您配置的方法顺序依次处理信号
2. 显示处理进度和使用的Pipeline
3. 构建数据集并训练模型

## 技术细节

### 通道数变化规则

| 配置 | 输入 | LCD 后 | FastICA 后 |
|------|------|--------|-----------|
| None | 1 通道 | - | - |
| LCD | 1 通道 | 10 通道 | - |
| FastICA* | 1 通道 | - | 跳过 |
| LCD → FastICA | 1 通道 | 10 通道 | 2 通道 |
| FastICA → LCD | 1 通道 | 跳过 | 10 通道 |

*注：FastICA 需要多通道输入，单通道时会自动跳过

### 处理流程示意

```
原始信号 (N,)
    ↓
[步骤 1: LCD]
    ↓
10 个 ISC 分量 (N, 10)
    ↓
[步骤 2: FastICA]
    ↓
2 个独立成分 (N, 2)
    ↓
[步骤 3: None] - 跳过
    ↓
[步骤 4: None] - 跳过
    ↓
保存为.mat 文件
```

## 控制台输出示例

```
Processing signal.npy -> processed_data/ica_results/signal_1234567890.mat
Processing pipeline: LCD -> FastICA

执行 LCD 分解...
LCD 完成，生成 10 个分量
执行 FastICA 分离...
FastICA 完成，输出 2 个成分
处理结果已保存至 processed_data/ica_results/signal_1234567890.mat
```

## 扩展示例

未来如果您添加了新的处理方法（如小波变换），只需：
1. 在 `PROCESSING_METHODS` 列表中添加新方法名称
2. 在 `process_signal_pipeline` 函数中添加对应处理逻辑
3. GUI 会自动显示新选项

## 向后兼容

如果您所有步骤都选择"None"，系统会使用默认的 LCD+FastICA 流程，保持与原版一致。
